Welcome

notifyr is a Firefox add-on which notifies you of new mail via a toast-like notification.

Installation

  1. Download notifyr.xpi and drag-and-drop it into your Firefox window
  2. Follow the on-screen instructions to install notifyr

Help and support

Just raise an issue at the Issues page and I'll be happy to help you!

About

notifyr is free & open-source (licensed under the Apache 2.0 license) and is hosted at GitHub.

Important: notifyr is custom built for a very specific set of users and not for the general public.
